Simple React app to show key React concepts.

Project goal: practice working with components.

To make the app work:
1. Clone the repository
2. Install Node.js
3. Open the project (VS Code or your fav editor)
4. Open local terminal (in the repository location .../ReactKeyConceptsApp>)
5. Run "npm install" command and wait
6. Run "npm start" command and enjoy the app

Project created with course:
